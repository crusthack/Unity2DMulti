# Unity2DMulti 
- 멀티플레이 게임을 직접 구현해보는 것에 초점을 둔 학습용 프로젝트입니다.
- 유니티를 체계적으로 배워보거나 게임을 만들어본 경험이 없어, 게임자체는 단순히 게임이 정상적으로 돌아가도록 최소한의 구현만 신경 썼습니다. 
- 기술 스택: unity, C#, IOCP, protobuf

## 싱글 플레이
- 로그인 없이 로컬에서 기본적인 게임 플레이
- 캐릭터 성장 없이, 몹 처치시의 점수 상승 및 몹 충돌시의 점수 하락만 존재합니다. 
- 스킬 시스템이 존재합니다.
- 게임 상태(점수, 위치)를 로컬에 저장해 불러올 수 있습니다. 

## 호스트 기반 멀티플레이
- 이름을 입력해 로그인을 합니다. 중복된 이름으로 이미 로그인된 유저가 있으면 거부됩니다. 
- 릴레이 서버에 접속해 다른 유저가 만든 게임 방에 접속하든가 직접 방을 만들어서 다른 유저와 게임할 수 있습니다.
- 릴레이서버는 단순 패킷 중계만 하며, 게임 로직은 호스트 유저의 컴퓨터에서 돌아갑니다. 
- 게임은 유저들 상태만 동기화하는 방식으로 최소한으로만 구현하였습니다.
