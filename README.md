# Unity2DMulti 
- 멀티플레이 게임을 직접 구현해보는 것에 초점을 둔 학습용 프로젝트입니다.
- 유니티를 체계적으로 배워보거나 게임을 만들어본 경험이 없어, 게임자체는 단순히 게임이 정상적으로 돌아가도록 최소한의 구현만 신경 썼습니다. 
- 기술 스택: unity, C#, IOCP, protobuf, ASP.Net Core

## 싱글 플레이
- 로그인 없이 로컬에서 기본적인 게임 플레이
- 뱀파이어 서바이벌 라이크 게임의 형식을 최소한으로 구현했습니다.
- 캐릭터 성장 없이, 몹 처치시의 점수 상승만 존재합니다. 
- 스킬 시스템이 존재합니다.
- 게임 상태(점수, 위치)를 로컬에 저장해 불러올 수 있습니다. 

## 호스트 기반 멀티플레이 서버
- 웹을 통해 회원가입 후 클라이언트에서 아이디를 입력해 로그인합니다. 이를 기반으로 유저를 식별합니다.
- 릴레이 서버에 접속해 다른 유저가 만든 게임 방에 접속하든가 직접 방을 만들어서 다른 유저와 게임할 수 있습니다.
- 릴레이서버는 단순 패킷 중계만 하며, 게임 로직은 호스트 컴퓨터에서 돌아갑니다.
